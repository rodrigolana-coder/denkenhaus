Seção 2 de 5
Gramática

1. Poser des questions pour demander des informations

Uma estrutura muito comum é:

Est-ce que + phrase

Est-ce que le petit déjeuner est inclus ?

Est-ce qu’il y a internet ?

2. Question avec combien

Combien é usado para perguntar quantidade ou preço.

Combien coûte une nuit ?

Combien de nuits ?

3. Réserver

Estrutura principal:

Je voudrais réserver + nom

Je voudrais réserver une chambre.

4. Révision

  • nombres para falar de preços e número de noites
  • coordonnées para dar informações pessoais numa reserva
  • où para perguntar localização
  • combien para perguntar preço ou quantidade
