Comment fonctionne le système postal en France ?
En France, le service postal s’appelle La Poste. C’est une organisation très importante dans le pays.
Les personnes utilisent La Poste pour envoyer des lettres, des cartes postales et des colis.
Pour envoyer une lettre, on écrit le nom du destinataire, son adresse complète et le code postal.
On doit aussi écrire les informations de l’expéditeur, c’est-à-dire la personne qui envoie la lettre.
Le code postal en France a cinq chiffres. Les deux premiers chiffres indiquent le département.
Les autres chiffres indiquent la ville ou une zone spécifique.
Par exemple, le code postal de Paris commence par 75. Chaque ville a un code postal différent.
Pour envoyer une lettre, on achète un timbre et on le colle sur l’enveloppe.
Ensuite, on met la lettre dans une boîte aux lettres.
Aujourd’hui, beaucoup de personnes utilisent Internet pour communiquer.
Mais La Poste reste très importante pour envoyer des documents officiels et des colis.
