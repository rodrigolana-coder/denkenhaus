Gramática — Les adjectifs et les noms de personnalité

1. Décrire avec un adjectif

Usamos être + adjectif para descrever a personalidade.

Il est patient.

Elle est généreuse.

Ils sont sympathiques.

2. Décrire avec un nom

Também podemos usar nomes abstratos para falar de qualidades e defeitos.

Il a beaucoup de patience.

Elle a beaucoup de courage.

Sa plus grande qualité est la gentillesse.

Son principal défaut est la timidité.

3. Donner son opinion

Para expressar opinião sobre alguém, podemos usar:

Je pense qu’il est très gentil.

Je trouve qu’elle est très sympathique.

À mon avis, il est très honnête.

Je crois qu’elle est très généreuse.

4. Deux façons de décrire une personne

Avec un adjectifAvec un nom
Il est généreux.Il a beaucoup de générosité.
Elle est très patiente.Elle a beaucoup de patience.
Il est honnête.Sa plus grande qualité est l’honnêteté.
Elle est timide.Son principal défaut est la timidité.
