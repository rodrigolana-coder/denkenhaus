How to give an opinion

A: What do you think of London? — B: I think it’s fantastic.

A: What do you think of this restaurant? — B: I think it’s very good.

A: Do you like your city? — B: Yes. I think it’s great.

A: Is there a museum? — B: Yes, there is.

Section 2 of 5
Grammar

1. I think…

Use I think… to give your opinion.

I think London is beautiful.

I think this café is great.

I think the food is delicious.

I think this park is very nice.

2. I don’t think…

Use I don’t think… to give a negative opinion politely.

I don’t think it’s expensive.

I don’t think this restaurant is very good.

I don’t think this hotel is nice.

3. There is / There are

Singular

There is a museum.

There is a beautiful park.

There is a famous palace.

Plural

There are some cafés.

There are a lot of restaurants.

There are some museums.

4. Some

Use some with plural nouns.

There are some parks.

There are some restaurants.

There are some hotels.

5. A lot of

Use a lot of to talk about a large quantity.

There are a lot of tourists.

There are a lot of cafés.

There are a lot of museums.

There are a lot of beautiful places.
