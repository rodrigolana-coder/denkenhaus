Seção 2 de 5
Gramática

1. Passé composé — révision

Structure :

avoir / être + participe passé

Elle a changé.

Nous avons vu l’exposition.

Je suis allé au musée.

2. Avoir l’air

👉 impression.

Il a l’air calme.

Elle a l’air heureuse.

Tu as l’air fatigué.

3. Être x avoir l’air

Être = caractéristique réelle.

Il est calme.

Elle est sportive.

Avoir l’air = impression.

Il a l’air calme.

Elle a l’air sportive.

4. C’est / Il est

C’est = identification.

C’est mon cousin.

C’est une actrice célèbre.

Il est / Elle est = description.

Il est sportif.

Elle est élégante.

5. Verbes de perception

VerbeSens
voirver
regarderolhar
entendreouvir
écouterescutar

touchertocar
goûterprovar

voir ≠ regarder

entendre ≠ écouter
