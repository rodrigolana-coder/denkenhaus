How to ask about prices

A: How much is this notebook? — B: It’s £3.50.

A: How much is that umbrella? — B: It’s £15.

A: How much are these flowers? — B: They’re £9.99.

A: I’ll take it. — B: Thank you.

Section 2 of 5
Grammar

1. THIS / THAT

Use this for something near you.

Use that for something far from you.

This book is new.

That umbrella is blue.

This café is nice.

That shop is expensive.

2. THESE / THOSE

Use these for plural things near you.

Use those for plural things far from you.

These flowers are beautiful.

Those books are cheap.

These shoes are new.

Those cars are expensive.

3. Questions with this / that / these / those

How much is this book?

How much is that phone?

How much are these flowers?

How much are those shoes?

4. Review — Colours

This bag is black.

That umbrella is green.

These flowers are pink.

Those cars are red.
