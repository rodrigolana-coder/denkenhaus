Seção 2 de 5
Gramática

1. Décrire une personne

Il est grand.

Elle est élégante.

Il a les cheveux noirs.

Elle a les yeux verts.

2. Le passé composé pour les changements

👉 usado para mudanças.

Il a changé.

Elle a coupé ses cheveux.

Il a perdu du poids.

Elle a commencé le sport.

3. Avoir l’air

👉 impressão / aparência momentânea.

avoir l’air + adjectif

Il a l’air fatigué.

Elle a l’air heureuse.

Tu as l’air stressé.

4. Être x avoir l’air

Être = característica mais objetiva.

Il est calme.

Elle est sportive.

Avoir l’air = impressão.

Il a l’air calme.

Elle a l’air sportive.

5. C’est x il est / elle est

C’est = identificação.

C’est mon frère.

C’est une actrice célèbre.

C’est un étudiant.

Il est / Elle est = descrição.

Il est sportif.

Elle est élégante.

Il est médecin.

6. Comparaison

C’est un acteur célèbre. / Il est célèbre.

C’est mon ami. / Il est gentil.
