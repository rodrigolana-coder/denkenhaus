Seção 2 de 5
Gramática — Les pronoms personnels COD

1. Le / La / Les

Os pronomes COD substituem um objeto direto.

LE substitui um nome masculino singular.

Tu aimes le pantalon ? → Oui, je le trouve élégant.

LA substitui um nome feminino singular.

Tu aimes la veste ? → Oui, je la trouve très belle.

LES substitui um nome plural.

Tu aimes les chaussures ? → Oui, je les adore.

2. Position

O pronome vem antes do verbo.

Je porte la veste. → Je la porte.

J’aime les baskets. → Je les aime.

Il achète le manteau. → Il l’achète.

3. Avec les verbes d’opinion

Muito usado com trouver, aimer e adorer.

Je le trouve élégant.

Je la trouve originale.

Je les trouve confortables.

4. Attention à l’apostrophe

Antes de vogal, le e la viram l’.

Je aime la chemise. ❌

Je l’aime. ✅

Je achète le manteau. ❌

Je l’achète. ✅
