Seção 2 de 5
Gramática

1. Demostrativos

👉 este / esta / estos / estas

masculino singular → este

Este museo es interesante.

femenino singular → esta

Esta ciudad es bonita.

plural → estos / estas

Estos monumentos son increíbles.

2. Hablar de las vacaciones

👉 pasar vacaciones

Paso buenas vacaciones.

Estoy pasando unas vacaciones increíbles.

3. Dar opiniones

👉 ser + adjetivo

Es muy bonito.

Es interesante.

Son increíbles.

4. El clima

👉 hacer + calor / frío / buen tiempo

Hace calor.

Hace buen tiempo.
