Seção 2 de 5
Gramática

1. Passé composé — rappel

👉 estrutura:

avoir / être + participe passé

J’ai travaillé.

J’ai voyagé.

Je suis né.

2. Verbes fréquents pour raconter une vie

être né → je suis né

commencer → j’ai commencé

travailler → j’ai travaillé

voyager → j’ai voyagé

apprendre → j’ai appris

3. Structure pour raconter une vie

👉 modèle simple :

Je suis né...

J’ai étudié...

J’ai travaillé...

J’ai voyagé...

4. Exemple complet

Je suis né en 1995.

J’ai étudié à l’université.

J’ai travaillé dans une entreprise.

J’ai voyagé.
