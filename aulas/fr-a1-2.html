Gramática

1. Masculin / Féminin des nationalités

Généralement :

  • -ais → -aise : Français → Française
  • -ien → -ienne : Canadien → Canadienne
  • -ois → -oise : Chinois → Chinoise
  • -ain → -aine : Américain → Américaine
  • -ol → -ole : Espagnol → Espagnole
  • -e → identique : Russe → Russe
  • Cas spécial : Grec → Grecque / Turc → Turque

Je suis canadien. · Elle est française. · Je suis grecque.

2. Articles des pays

  • Pays féminins : la → la France, la Chine
  • Pays masculins : le → le Portugal, le Mexique
  • Pays pluriels : les → les États-Unis
  • Voyelle : l’ → l’Allemagne

la France · le Mexique · les États-Unis · l’Allemagne
